=================
## 不再维护了，仅做存档

angularjs封装的select2（支持任意版本的select2）

* 支持动态ng-model
* 支持select标签
* 支持自定义配置及多选（与select2原生的配置方式一致）
* 支持ajax
* 支持自定义内置配置

具体使用请下载源码，打开index.html查看。
